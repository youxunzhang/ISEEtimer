# ISEEtimer 功能文档

本文档用于说明 ISEEtimer（计时/倒计时工具站）的主要功能、页面入口与核心交互，便于维护者与内容运营快速了解站点能力与流量入口。

## 1. 产品概览

ISEEtimer 提供多种计时与倒计时场景，覆盖常用的倒计时、专注番茄钟、秒表、节日/生日/考试倒计时、世界时间工具、呼吸训练等内容页，适用于工作学习计时、节日提醒、教学/展示与内容分享。

## 2. 页面与功能一览

> 建议优先引流的高关注入口：计时器首页、节日倒计时、世界时间工具、全屏倒计时、30 分钟倒计时、考试/生日/婚礼倒计时等。

### 2.1 首页 / 计时器聚合（`index.html`）
- **计时器**：正向计时、开始/暂停/重置、小时/分钟/秒设置。
- **倒计时**：输入倒计时时间、到时提醒。
- **番茄钟**：工作/休息周期、轮次提示、进度展示。
- **秒表**：毫秒级计时、开始/暂停/重置。
- **主题切换**：浅色/深色主题。
- **导航入口**：跳转到多个子功能页面。

### 2.2 30 分钟倒计时（`30-minute-countdown.html`）
- **固定 30 分钟倒计时**，适合专注/课堂/会议场景。

### 2.3 全屏倒计时（`fullscreen-countdown.html`）
- **全屏展示**，适合投屏或课堂/会议使用。

### 2.4 节日倒计时聚合（`festival-countdown.html`）
- **多节日倒计时列表**，可作为节庆流量入口。

### 2.5 节日/主题倒计时页面
- `birthday-countdown.html`：生日倒计时。
- `exam-countdown.html`：考试倒计时。
- `wedding-countdown.html`：婚礼倒计时。
- `holiday-background-countdown.html`：节日背景主题倒计时。
- `featured-countdowns.html`：精选倒计时集合。

### 2.6 世界时间工具（`world-time-tools.html`）
- **多时区时间对比**，适合跨时区协作与国际化场景。

### 2.7 北京时间（`beijing-time.html`）
- **标准时间展示**，用于对时或直播/活动提示。

### 2.8 子午流注时钟（`ziwu-liuzhu-clock.html`）
- **传统时辰显示与养生提示**，内容型入口。

### 2.9 计时类小游戏/互动
- `game.html`：计时小游戏入口。
- `countdown-game.html`：倒计时挑战玩法。

### 2.10 训练与实用工具
- `breathing-exercise.html`：呼吸练习节奏引导。
- `incense-timer.html`：香薰/冥想/禅修类计时。

### 2.11 文章与内容页
- `articles.html`：文章聚合页。
- `article-time-planning.html`：时间规划文章。
- `article-website-planning.html`：网站规划文章。
- `how-to-create-countdown-timer.html`：倒计时教程。
- `free-ai-tools.html`：工具推荐类内容。

## 3. 通用功能与交互

### 3.1 导航与布局
- 顶部导航提供多个功能入口。
- 统一样式与响应式布局，适配桌面与移动端。

### 3.2 计时核心逻辑
- 计时器、倒计时、番茄钟、秒表共享核心控制逻辑。
- 倒计时结束提供提示与状态停止。

### 3.3 主题与性能优化
- 支持主题切换与偏好持久化。
- 使用 Web Worker 提升计时精度与稳定性。

## 4. 关键脚本与样式文件

| 文件 | 作用 |
| --- | --- |
| `script.js` | 计时器/倒计时/番茄钟/秒表/主题切换等核心逻辑 |
| `timer-worker.js` | 后台计时 Web Worker |
| `styles.css` | 全站通用样式 |
| `dropdown.js` / `footer.js` / `articles.js` | 页面交互与内容辅助脚本 |

## 5. 使用说明（快速上手）

1. 打开 `index.html` 体验完整计时器功能。
2. 按需跳转至功能页（如 `festival-countdown.html`、`world-time-tools.html`）。
3. 若用于引流推荐，将高热度入口加入主站导航或首页推荐卡片。

## 6. 运营与扩展建议

- **流量入口优先级**：节日倒计时、考试/生日/婚礼倒计时、全屏倒计时、世界时间工具。
- **内容扩展**：增加更多节日/主题倒计时页面，复用现有结构与样式。
- **国际化**：为高流量页面提供英文版本以扩大受众。
